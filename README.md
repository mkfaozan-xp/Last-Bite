# LastBite - Surplus Food Platform

A highly advanced, feature-rich web application for saving surplus food, fighting food waste, and helping those in need.

## 🌟 Key Features

### Global Features
- **Currency**: Indian Rupees (₹)
- **Dark/Light Mode**: Full theme toggle support
- **Premium UI**: Glassmorphism, gradients, blur effects
- **Smooth Animations**: Motion animations throughout
- **Responsive Design**: Works on all devices

### User Types
1. **Customer Dashboard** - Main food ordering interface
2. **Restaurant Dashboard** - Inventory and earnings management
3. **NGO Dashboard** - Donation management

### Customer Features

#### 🎯 Stock Market Food System
- Set target prices for food items
- View price history graphs (like stock market)
- Get notified when prices drop
- Auto-order when price targets are met

#### 🎮 Gamification
- **Guess the Leftover**: Predict which food will remain unsold
- **Daily Spin**: Win discounts and rewards
- **Leaderboard**: Compete with other food savers
- **Badges & Achievements**: Earn rewards for saving food

#### 💰 Wallet System
- Add money to wallet
- Pay using wallet balance
- Cashback system
- Transaction history
- Send gift cards to friends

#### 🚂 Train Delivery
- Special feature: Deliver food to trains
- Enter train number, seat, and station
- Track delivery to your seat

#### 🌱 Impact Dashboard
- Track meals saved
- CO₂ emissions reduced
- Money saved
- Personal contribution stats
- Environmental impact visualization

#### 🔍 Advanced Search & Filters
- Filter by Veg/Non-veg
- Price range slider
- Distance filter
- Category filters (Fast Food, Indian, Dessert, Healthy)
- Real-time search with suggestions

#### ⏰ Live Deals
- Real-time countdown timers
- Stock availability indicators
- Live pricing updates
- High-quality food images
- Up to 70% discounts

### Restaurant Dashboard Features
- **Earnings Analytics**: Daily, monthly, and total earnings
- **Inventory Management**: Batch-wise tracking
- **Expiry Countdown**: Track food expiration
- **Dynamic Pricing**: Update prices in real-time
- **AI Demand Prediction**: Smart forecasting charts
- **Order Management**: Track and manage orders
- **Upload Items**: Add new food items to platform

### NGO Dashboard Features
- **Donation Management**: Accept/reject donations
- **Pickup Scheduling**: Coordinate pickup times
- **Contact System**: Email and phone integration
- **Real-time Notifications**: Get instant alerts
- **Impact Tracking**: Meals served and lives impacted
- **Partner Network**: Manage restaurant partnerships

### Additional Features
- **Order Tracking**: Real-time order status
- **Multiple Payment Methods**: UPI, Cards, Net Banking
- **Profile Management**: Edit personal information
- **Subscription Plans**: Gold and Platinum memberships
- **Trending Searches**: Discover popular items
- **Restaurant Ratings**: User reviews and ratings
- **Multi-language Support**: Ready for expansion
